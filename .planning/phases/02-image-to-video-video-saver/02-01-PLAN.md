Add Image-to-Video (I2V) generation capability to the Seedance 2.0 ComfyUI plugin.

Purpose: Users can provide a reference image (ComfyUI IMAGE tensor) and a prompt to generate a video that uses the image as visual context. This is the second generation mode after text-to-video.

Output: `create_i2v_video` and `create_and_wait_i2v` functions in `api_client.py`, `SeedanceImageToVideo` node class in `nodes.py`, updated registration in `__init__.py`, and comprehensive unit tests.
